API Access

You can make a GET request to /api/<isbn> to get some information about the books. <isbn> is the ISBN number of the book.

Response will be a JSON object containing the book's title, author, publication date, ISBN number, review count and average score.

An example JSON object returned: 

            
    {
        "author": "Richard Yates",
        "average_score": 4.5,
        "isbn": "0413757102",
        "review_count": 2,
        "title": "Revolutionary Road",
        "year": 1961,
    }

If the requested ISBN number isnâ€™t found, a 404 error is returned.
